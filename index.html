Movie Search GET
Search for movies by title. Returns a list of matching results including title, year, rating, and metadata.
https://web-production-cc716.up.railway.app/movies/search
Name Type Required Description
title string required The movie title (or partial title) to search for. Case-insensitive partial matches are supported.
ℹ️ Requests are sent directly to the live API. Enter a movie title below and click Send Request to see the real JSON response.
Response
Response will appear here

The API returns a JSON array of movie objects. Each item in the array represents a matching movie.

[ { "id": integer // Unique movie identifier "title": string // Full movie title "year": integer // Release year "genre": string // Primary genre "rating": float // Score out of 10 "director": string // Director name (if available) "description": string // Short synopsis } ]
